소개

이 워크샵을 통해 AWS에서 SAP HANA 를 설치하고 운영하는 방법에 대해 알아봅니다.

준비사항

  • AWS 계정: VPC, EC2, S3 등의 자원을 생성할 수 있는 권한이 필요합니다.
  • AWS 리전: 이번 실습은 버지니아 (us-east-1) 리전에서 실행합니다.
  • SAP HANA 설치파일: SAP HANA 설치 파일을 S3 Bucket에 사전에 업로드 하셔야 합니다.
  • 브라우저: 최신 버전의 크롬, 파이어폭스를 사용하세요.

유의사항

  • 실습1. Launch Wizard for SAP HANA, 실습1. SAP HANA Quick Starts 두 개중 하나만 수행하시기 바랍니다.
  • 실습중 제공되는, SAP HANA 소프트웨어 S3 URL 은 실습중에만 사용이 가능합니다.
